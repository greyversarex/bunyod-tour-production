Необходимо полное налаживание системы гида:
1) Для гидов необходимо сделать страницу, на которую попадаешь после нажатия на карточку гида на странице тургид. На должно быть его описание и отзывы и основная информация о гиде (имя и фамилия, опыт, оценка и т.д.); Регион обслуживания гида, город(а), страна (страна или город(а), где гид предоставляет услуги); - Это те же города и страны которые указываются при создании гида. Описание гида есть при его создании через админ-панель, в модальном окне создания тура есть поля Комментарии по гиду, это и есть описание.
2) Надо наладить систему рейтинга или оценки гида от туристов. Сделай страницу для оставления отзыва по гиду. У нас уже реализовано страница для сбора отзыва по туру. Надо что-то типо неё. Я тебе прикрепил фото к этому сообщению с примерно того, какой должна быть форма сбора отзыва. Для каждого тургида должна быть ссылка для оставления отзыва конкретно по отношению к нему. Турист заполняет форму, отправляет оцену с отзывом, и она должна попадать на модерацию в админ панель, а после одобрения уде браться в учет и попадать на страницу гида. У нас есть рейтинговая система, по ней должна показываться оценка туриста с учетом всех оценок, думаю ты понял о чем я, этим тоже займись. Оценка отображается на карточке гида, можешь посмотреть коды всех страниц относящих к гиду для большего понимания всего.
3) Кабинет гида. У нас уже налажено так, что при завершении тура в кабинете гиба появилась кнопка Собрать отзыв, которая отправляет ссылку на страницу для заполнения отзыва на тур. Нам необходимо добавить еще одну кнопку – «Отзыв гида» для отправки ссылки на страницу с формой заполнения отзыва и оценки по гиду. Можешь переименовать кнопку Собрать отзывы на Отзывы по туру.
